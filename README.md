# 米拉奇战记GM修改版 Miragine_War
米拉奇战记（Miragine War）是一款快节奏的横向出兵对战策略小游戏，选择兵种、控制出兵节奏与资源增长，在推进战线的同时摧毁对方水晶获胜；单位有轻甲/重甲与不同射程、攻速和伤害档位，克制关系明确，上手简单但很上头。支持双人同屏对战，还能和朋友现场互相博弈、互相坑一把。

略略略修改版（GM修改版）说明：游戏内按G打开GM面板（可滚动），可分别配置左右双方的开局金钱/收入/上限、人力上限、出兵间隔、水晶血量/伤害/射程/攻速、全局子弹速度与全局子弹爆裂时间、单位倍率等；支持“应用到本局(⇧⏎)”/“应用并重开(⇧R)”/“恢复原版默认并重开(⇧⌫)”。你可以把它当成“自定义对战器”：和朋友左右两边各自设定属性，现场开一局自定义属性对决；想爽玩也可以直接把我方拉满、对方削弱。
